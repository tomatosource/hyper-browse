# hyper-brwose

[Hyper](https://hyper.is/) plugin to display URLs - intended for development uses primarily.


## Installation

[Install Hyper](https://hyper.is/#installation) and add `hyper-browse` to `plugins`
in `~/.hyper.js`.

## Usage

### Opening

- ```$ browse:https://google.com```

### Closing

- `Ctrl + C`
